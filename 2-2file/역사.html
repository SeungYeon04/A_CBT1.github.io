리눅스 역사 주요 인물

리누스 토르발스 (Linus Torvalds)

역할: 리눅스 커널 창시자
주요기여: 1991년 리눅스 커널 개발 후 오픈소스로 공개
관련 명령어: uname 시스템 정보 출력. 리눅스 커널 확인 유용
uname -a

리처드 스톨만 (Richard Stallman)

역할: GNU 프로젝트 창시자. (자유 소프트웨어 운동의 선구자.)
주요기여: GNU 툴체인 개발. GPL 라이선스 작성
관련 명령어: gcc (GNU Compiler Collection, C컴파일러)
gcc -o outputfile sourcefile.c

앨런콕스 (Alan Cox)

역할: 리눅스 커널 개발자
주요 기여: 리눅스 커널 네트워킹 스택 개발.
관련 명령어: ifconfig (네트워크 인터페이스 설정)
ifconfig eth0 up

마크 셔틀워스 (Mark Shuttleworth)

역할: 우분투 배포판 설립자.
주요 기여: 우분투 배포판 통해 리눅스 접근 쉬워지게 함.
관련 명령어: apt-get (패키지관리 명령, 소프트웨어설치제거)
sudo apt-get update

앤드류 모턴 (Andrew Morton)

역할: 리눅스 커널 유지 관리자.
주요 기여: 커널 패치, 테스트 작업
관련 명령어: patch (파일에 패치를 적용)
patch < patchfile

브라이언 커니한 (Brian Kernighan)

역할: 유닉스 창시자 중 한 명, C언어 공동 개발자
주요기여: 유닉스 철학과 도구의 개발
관련 명령어: awk (텍스트 프로세싱 언어)
awk '{print $1}' filename

데니스 리치 (Dennis Ritchie)

역할: 유닉스 창시자 중 한 명, C언어 개발자
주요기여: 유닉스 운영체제와 C언어 개발
관련 명령어: cc (C컴파일러 gcc이전의 옛 기본 컴파일러)
cc -o outputfile sourcefile.c

마이클 와이드너스 (Michael Widenius)

역할: MySQL의 공동 창시자.
주요기여: 오픈소스 관계형 데이터베이스 MySQL개발.
관련 명령어: mysql (MySQL 데이터베이스와 상호작용)

이언 머독 (Ian Murdock)

역할: 데비안 프로젝트 창시자
주요기여: 데비안 리눅스 배포판 개발.
관련 명령어: dpkg (데비안 패키지 관리 명령어 - 오프라인)
dpkg -i package.deb

래리 월 (Larry Wall)

역할: Perl 프로그래밍 언어 창시자.
주요 기여: 텍스트 처리와 시스템 관리에 강력한 Perl 언어 개발.
관련 명령어: perl (Perl 스크립트를 실행)
perl script.pl

리눅스 편집기 인물

Vi = 빌 조이 (Bill Joy)

역할: vi 편집기 창시자.
주요기여: Unix 시스템에서 Vi편집기를 개발하여 기본편집기로 자리 잡음.
관련 명령어 vi
vi filename

GNU Emacs (리처드 스톨만)

역할: Emacs 텍스트 편집기 개발자.
주요 기여: 확장 가능한 텍스트 편집기 Emacs개발
관련 명령어: emacs(텍스트 편집기)
emacs filename

Bram Moolenaar

역할: Vi확장판 Vim 개발.

James Gosling (제임스 고슬링)

역할: 고슬링 에디터(vi 초기버전 중 하나)와 JAVA언어 창시자.
주요기여: 고슬링에디터, Java개발
관련 명령어: javac (Java컴파일러)
javac MyClass.java

리눅스 역사 년도별 순서

주요 사건 및 인물 활동 연대기
1960년대
1969년: 데니스 리치(Dennis Ritchie)와 켄 톰슨(Ken Thompson)이 벨 연구소에서 유닉스(Unix) 운영체제 개발 시작.
1970년대
1970년: 유닉스 첫 번째 버전 출시.
1973년: 데니스 리치와 브라이언 커니한(Brian Kernighan)이 C 프로그래밍 언어 개발.
1976년: 빌 조이(Bill Joy)가 Vi 편집기 개발 시작.
1979년: Vi 편집기 첫 공개.
1980년대
1983년: 리처드 스톨먼(Richard Stallman)이 GNU 프로젝트 시작, 자유 소프트웨어 재단(FSF) 설립.
1985년: 리처드 스톨먼이 GNU Emacs 개발.
1987년: 래리 월(Larry Wall)이 Perl 프로그래밍 언어 개발.
1989년: 리처드 스톨먼이 GNU GPL (General Public License) 첫 버전 발표.
1990년대
1991년: 리누스 토르발스(Linus Torvalds)가 리눅스 커널 첫 버전 발표.
1993년: 이언 머독(Ian Murdock)이 데비안 프로젝트 시작.
1994년: 리눅스 커널 1.0 버전 출시.
1995년: 마이클 와이드너스(Michael Widenius)가 MySQL 데이터베이스 시스템 개발 시작.
1996년: Bram Moolenaar가 Vim 텍스트 편집기 공개.
2000년대
2004년: 마크 셔틀워스(Mark Shuttleworth)가 우분투(Ubuntu) 첫 버전 출시.
2006년: 리누스 토르발스와 기타 커널 개발자들이 리눅스 재단(The Linux Foundation) 설립.
2010년대
2014년: 리눅스 커널 3.x 시리즈 출시.
2015년: 리눅스 커널 4.x 시리즈 출시.
2019년: 리눅스 커널 5.x 시리즈 출시.

리눅스 관련 주요 명령어

1. uname
사용법: uname -a
설명: 시스템 정보를 출력합니다.
관련 인물: 리누스 토르발스

2. gcc
사용법: gcc -o outputfile sourcefile.c
설명: GNU Compiler Collection, C 컴파일러.
관련 인물: 리처드 스톨먼

3. ifconfig
사용법: ifconfig eth0 up
설명: 네트워크 인터페이스 설정.
관련 인물: 앨런 콕스

4. apt-get
사용법: sudo apt-get update
설명: 패키지 관리 명령어, 우분투에서 소프트웨어 설치/제거에 사용.
관련 인물: 마크 셔틀워스

5. patch
사용법: patch < patchfile
설명: 파일에 패치를 적용.
관련 인물: 앤드류 모턴

6. awk
사용법: awk '{print $1}' filename
설명: 텍스트 프로세싱 언어.
관련 인물: 브라이언 커니한

7. mysql
사용법: mysql -u username -p
설명: MySQL 데이터베이스와 상호작용.
관련 인물: 마이클 와이드너스

8. dpkg
사용법: dpkg -i package.deb
설명: 데비안 패키지 관리 명령어.
관련 인물: 이언 머독

9. perl
사용법: perl script.pl
설명: Perl 스크립트를 실행.
관련 인물: 래리 월

10. vi
사용법: vi filename
설명: 텍스트 편집기.
관련 인물: 빌 조이

11. vim
사용법: vim filename
설명: 텍스트 편집기.
관련 인물: Bram Moolenaar
