LibreOffice 주요 도구

LibreOffice Writer

설명: 문서 작성 도구로 Microsoft Word와 유사합니다.
기능: 텍스트 문서 작성, 서식 설정, 표 삽입, 이미지 삽입 등.
명령어: libreoffice --writer filename.odt

LibreOffice Calc(Calc = 계산기)

설명: 스프레드시트 애플리케이션으로 Microsoft Excel과 유사합니다.
기능: 데이터 분석, 차트 생성, 수식 계산, 피벗 테이블 등.
명령어: libreoffice --calc filename.ods

LibreOffice Impress

설명: 프레젠테이션 도구로 Microsoft PowerPoint와 유사합니다.
기능: 슬라이드 작성, 애니메이션, 템플릿 사용 등.
명령어: libreoffice --impress filename.odp

LibreOffice Draw

설명: 벡터 그래픽 및 다이어그램 도구.
기능: 플로우차트, 다이어그램, 포스터 제작 등.
명령어: libreoffice --draw filename.odg

LibreOffice Base

설명: 데이터베이스 관리 도구로 Microsoft Access와 유사합니다.
기능: 데이터베이스 생성 및 관리, SQL 쿼리 작성 등.
명령어: libreoffice --base filename.odb

LibreOffice Math

설명: 수식 편집기.
기능: 수학적 수식 및 과학적 표기법 작성.
명령어: libreoffice --math filename.odf

주요 프로그램

텍스트 편집기

gedit: GNOME 데스크탑의 기본 텍스트 편집기입니다.
Kate: KDE 데스크탑 환경의 고급 텍스트 편집기입니다.
Leafpad: 가볍고 빠른 텍스트 편집기입니다. LXDE에서 자주 사용됩니다.

파일 관리자

Nautilus: GNOME 데스크탑의 기본 파일 관리자입니다.
Dolphin: KDE 데스크탑 환경의 파일 관리자입니다.
Thunar: Xfce 데스크탑 환경의 파일 관리자입니다.
PCManFM: LXDE 데스크탑 환경의 파일 관리자입니다.

웹 브라우저

Firefox: 리눅스에서 가장 많이 사용되는 웹 브라우저 중 하나입니다.
Chromium: 구글 크롬의 오픈 소스 버전입니다.
Epiphany (GNOME Web): GNOME 데스크탑 환경의 기본 웹 브라우저입니다.

터미널 에뮬레이터

GNOME Terminal: GNOME 데스크탑 환경의 기본 터미널입니다.
Konsole: KDE 데스크탑 환경의 기본 터미널입니다.
Xfce Terminal: Xfce 데스크탑 환경의 터미널입니다.
LXTerminal: LXDE 데스크탑 환경의 터미널입니다.

엑셀 및 계산기

LibreOffice Calc: LibreOffice 스위트의 스프레드시트 애플리케이션으로, Microsoft Excel과 유사합니다.
Gnumeric: 가볍고 빠른 스프레드시트 애플리케이션입니다.
KCalc: KDE 데스크탑 환경의 기본 계산기입니다.
gnome-calculator: GNOME 데스크탑 환경의 기본 계산기입니다.

그림 및 그래픽 편집기

GIMP (GNU Image Manipulation Program): 강력한 이미지 편집 소프트웨어입니다.
Inkscape: 벡터 그래픽 편집 소프트웨어로, Adobe Illustrator와 유사합니다.
Krita: 주로 디지털 페인팅에 사용되는 그래픽 편집 소프트웨어입니다.

멀티미디어

VLC Media Player: 다양한 오디오 및 비디오 포맷을 지원하는 미디어 플레이어입니다.
Rhythmbox: GNOME 데스크탑 환경의 음악 플레이어입니다.
Amarok: KDE 데스크탑 환경의 음악 플레이어입니다.

LibreOffice 애플리케이션

LibreOffice Writer: 문서 작성 도구.
LibreOffice Calc: 스프레드시트 애플리케이션.
LibreOffice Impress: 프레젠테이션 도구.
LibreOffice Draw: 벡터 그래픽 및 다이어그램 도구.
LibreOffice Base: 데이터베이스 관리 도구.
LibreOffice Math: 수식 편집기.
