ls 뒤에 명령어 설명
ls -c = 파일의 상태 변경시간을 기준으로 정렬하거나 표시
ls -s = 파일의 블록 크기를 표시
ls -l = 파일에 대한 상세 정보를 표시합니다.
ls -t = 파일을 수정시간 기준으로 정렬
ls -a = 숨김 파일을 포함한 모든 파일을 표시
ls -r = 파일 목록을 역순으로 정렬
ls -h = 파일 크기를 사람이 읽기 쉬운 형식 (KB, MB..)으로 표시

사용예시
ls -ltc = 파일을 상태변경시간 기준으로 정렬하고 상세정보 표시
ls -ls = 블록 크기와 함께 파일 목록을 표시
ls -lact = 숨김파일을 포함한 모든 파일을
상태변경 기준으로 정렬하고 상세정보표시

사용자권한


권한비트
r: 읽기권환
w: 쓰기권한
x: 실행권한

사용자유형
u: 파일의 소유자
g: 파일의 소유자가 속한 그룹
o: 다른 사용자(other)
a: 모든 사용자

조합예시
chmod o+x filename = 다른 사용자에게 실행권한 부여
chmod a+rwx filename = 모든 사용자에게 읽기 쓰기 실행 부여
chmod g-r filename = 그룹의 읽기권한 제거

특수권한 조합예시
chmod u+s filename = 파일이 소유자의 권한으로 실행되도록 설정
chmod g+s filename = 파일이 그룹 소유가의 권한으로 실행되도록 설정
chmod +t directoryname =
디렉터리파일은 파일소유자, 디렉터리소유자, root사용자만 삭제 가능(sticky bit)

소유권 변경

파일의 소유권 바꾸는 건 chown이다.
파일의 그룹 소유권을 바꾸는 건 chgrp

그룹소유권 변경 사용예시
chgrp 그룹이름 파일이름
chgrp ihr lin/a.txt = 파일 lin/a.txt의 그룹소유권을 ihr로 변경

소유권 변경 사용예시
chown 사용자이름 파일이름
chown newuser a.txt = 파일 a.txt의 소유권을 newuser로 변경

소유권 그룹소유권 둘 다 변경
chown 사용자이름:그룹이름 파일이름
chown newuser:newgroup a.txt =
a.txt파일의 소유권 사용자 newuser
그룹사용자newgroup으로 변경
