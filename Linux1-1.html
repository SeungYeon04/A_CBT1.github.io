시스템 부팅 및 초기화 + GRUB설정 업데이트

(GRUB) 부트로더 설정
문제: GRUB2의 설정파일을 수정하여
기본부팅항목을 변경하는 법을 설명하세요

설명:
a. GRUB 설정 파일을 엽니다.
sudo nano /etc/default/grub

b. GRUB_DEFAULT 항목을 수정하여
원하는 부팅 항목으로 변경
GRUB_DEFAULT=0

c. 변경사항 저장하고 나옴

d. GRUB 설정을 업데이트하여 변경사항 적용
sudo update-grub

사용자 추가 및 비번 설정

문제: 새로운 사용자 추가하고 비번 설정 설명하시오
설명:
a. 새로운 사용자 추가
sudo useradd -m newuser

b. 비번 설정
sudo passwd newuser

LVM 설정 및 관리

문제: LVM을 사용하여 논리볼륨을 생성하고 확장하는 법을 설명하시오
설명:
a.물리볼륨 생성
sudo pvcreate /dev/sdx1

b. 볼륨 그룹 생성
sudo vgdreate myvg /dev/sdx1

c. 논리볼륨 생성
sudo lvcreate -n mylv -L 10G myvg

d. 논리볼륨 확장
sudo lvextend -L +5G /dev/myvg/mylv

e. 파일시스템 확장
sudo resize2fs /dev/myvg/mylv

파일 시스템 생성 및 마운트

문제: ext4 파일 시스템을 생성하고 마운트하는 법 설명
설명:
a. ext4 파일 생성
sudo mkfs.ext4 /dev/sdx1

b. 파일 시스템을 마운트
sudo mount /dev/sdx1 /mnt
